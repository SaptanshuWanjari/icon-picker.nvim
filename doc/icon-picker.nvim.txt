*icon-picker.nvim*  Telescope icon picker for React projects

INTRODUCTION ~
  icon-picker.nvim provides a Telescope picker for icons from:
  - lucide-react
  - react-icons
  - @iconify-json/* with @iconify/react

FEATURES ~
  - Inserts JSX icon at cursor
  - Adds import safely
  - Handles multiline import blocks
  - Optional terminal preview

SETUP ~
>
  require("icon_picker").setup({
    command = "IconPicker",
    keymaps = {
      normal = "<leader>ii",
      insert = "<C-g>i",
    },
  })
<

COMMANDS ~
  :IconPicker
      Open picker.

MAPPINGS ~
  Configure through setup(). No default keymaps unless provided.

IN-PICKER ~
  <C-t>  Reopen source selector when multiple sources are available.

ICONIFY ~
  Iconify support reads installed @iconify-json/{prefix}/icons.json packages
  from node_modules. Selecting an Iconify icon inserts:
>
  <Icon icon="prefix:name" />
<
  and adds:
>
  import { Icon } from "@iconify/react";
<
